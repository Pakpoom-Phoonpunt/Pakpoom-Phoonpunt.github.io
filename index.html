Attributes

การใส่ Attributes จะใส่ที่ tag เปิดเเละใส่เป็นคู่ name="value" นะจ้ะ

<p name="value">Element</p>

ผมเป็น Hyperlink มี Attributes คือ href เเละ value คือ https://steamcommunity.com/id/volkzap/

TT ผมไม่มี Attributes (ข้อความที่ไม่มี Attributes)

ผมมี Attributes นะผมใช้ style value ผมคือ color: blue;

ผมก็มีลองเอาเมาส์มาชี้ทับผมสิ

Headings

Paragraphs

Paragraph เป็นอีก tag ที่ใช้เยอะมากใน HTML ใช้เขียนข้อความหรือบทความต่างๆ
Paragraph นั้นไม่ว่าเราจะเว้นช่องว่างระหว่างข้อความมากเท่าไร(การขึ้นบรรทัดใหม่ด้วย)ในตัว code html ของเราเเต่เมื่อผ่านการ render จาก browser เเล้วช่องว่างที่เราเว้นไว้เยอะมันจะกลายเป็นเเค่ช่อง spacebar ช่องเดียวอยู่ดี

<p>Element</p>

ผมเป็น Paragraph หล่ะ


เเต่เนื่องจากบทความบางบทความเช่น บทกวี จำเป็นต้องมีย่อหน้าหรือการเว้นวรรคที่สวยงามเราจึงมี tag อีกหนึ่ง tag ที่จะเเสดงผลทุกอย่างใน tag
ไม่ว่าเราจะเว้นช่องว่างไว้เยอะขนาดไหนก็ตามนั้นก็คือ tag <pre>

<pre>Element</pre>
