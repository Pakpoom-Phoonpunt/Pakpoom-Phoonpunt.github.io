Formatting

เราสามารถ Format ข้อความต่างๆของเราได้ด้วยการใช้ tag ตามด้านล่างนี้
- ข้อความตัวหนาเราใช้ tag <b>
- ข้อความที่เราต้องการให้ความสำคัญเราใช้ tag <strong>
- ถ้าข้อความไหนเราอยากให้ตัวเอียงเราใช้ tag <i>
- ข้อความที่เราอยากให้ตัวเล็ก(ที่สุด)เราใช้ tag <small>
- ข้อความที่เราต้องการไฮไลต์เราใช้ tag <mark>
- ข้อความที่เราต้องการขีดฆ่าเราใช้ tag <del>
- ข้อความที่เราต้องการขีดเส้นใต้เราจะใช้ tag <ins>
- ถ้าเราต้องการให้ข้อความเขียนอยู่ใต้ข้อความอื่นใช้ tag <sub>
- ถ้าเราต้องการให้ข้อความเขียนอยู่บนข้อความอื่นใช้ tag <sup>
หมายเหตุ ทุก tag ที่กล่าวมามี tag ปิดด้วยนะจ้ะ

CSS

Cascading Style Sheets (CSS) จะใช้เพื่อจัดโครงสร้างของ Webpage
เราสามารถเรียกใช้งาน CSS ได้ 3 วิธี ดังนี้

  • เเบบ Inline คือใช้ attribute style ใน tag เปิด
    <h1 style="color:blue;">Blue heading</h1>

  • เเบบ Internal ใช้ <style> ด้านใน <head> 
    		<head>
			<style>
				body {background-color: powderblue;}
				h1 {color: blue;}
			</style>
		</head>

  • แบบ External ใช้ <link> เพื่อดึงข้อมูลจากไฟล์ CSS มาใช้ในไฟล์ HTML 
    				<head>
					<link rel="stylesheet" href="styles.css">
				</head>

Tables

สำหรับการสร้างตาราง HTML มี tag ที่ใช้คือ <table> เเละ tag ย่ออื่นๆอีกดังนี้

  • row เเต่ละ row จะใช้ <tr>
  • ส่วนหัวข้อใหญ่ของตารางใช้ <th>
  • เนื้อหาเเต่ละ cell ใช้ <td>

Simple 
		<table style="width:50%">
			<tr>
				<th>Firstname</th>
				<th>Lastname</th>
				<th>Age</th>
			</tr>
			<tr>
				<td>Jill</td>
				<td>Smith</td>
				<td>50</td>
			</tr>
			<tr>
				<td>Eve</td>
				<td>Jackson</td>
				<td>94</td>
				</tr>
		</table>
		
	ที่ head ผมใส่ border ตามนี้ 

	table, th, td {
		border: 1px solid black;
	}

Lists

tag ของการสร้าง list เราจะใช้ตามนี้

tag รายละเอียด
<ul> สร้าง list เเบบ unordered list
<ol> สร้าง list เเบบ ordered list
<il> สร้างรายการต่างๆใน list
<dl> สร้าง list เเบบ description list
<dt> สร้างรายการต่างๆใน description list
<dd> ใช้เพิ่มคำอธิบายของรายการใน description list

			<ul>
				<li>Coffee</li>
				<li>Tea</li>
				<li>Milk</li>
			</ul>

  • Milk

			<ol>
				<li>Coffee</li>
				<li>Tea</li>
				<li>Milk</li>
			</ol>
ผลลัพธ์ :
  1. Coffee
  2. Tea
  3. Milk

ตัวอย่าง description list 

			<dl>
				<dt>Coffee</dt>
				<dd>- black hot drink</dd>
				<dt>Milk</dt>
				<dd>- white cold drink</dd>
			</dl>

Classes

Class ใน HTML ใช้ระบุ style ต่างๆให้กับ Element
ตัวอย่าง 


		<!--ในส่วน HEAD-->
		<head>
		<style>
			.city {
			  background-color: tomato;
			  color: white;
			  border: 2px solid black;
			  margin: 20px;
			  padding: 20px;
			}
			</style>
		</head>
		<!--ในส่วน BODY-->
		<body>
			<div class="city">
			  <h2>London</h2>
			  <p>London is the capital of England.</p>
			</div>
		</body>
ผลลัพธ์

London

London is the capital of England.

		<!--ในส่วน HEAD-->

				  color: white;

				}
				.middle{
					text-align: center;
				}
				</style>
			</head>
			<!--ในส่วน BODY-->
			<body>
				<div class="city middle">
				  <h2>London</h2>
				  <p>London is the capital of England.</p>
				</div>
			</body>
