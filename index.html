Attributes

การใส่ Attributes จะใส่ที่ tag เปิดเเละใส่เป็นคู่ name="value" นะจ้ะ

<p name="value">Element</p>

ผมเป็น Hyperlink มี Attributes คือ href เเละ value คือ https://steamcommunity.com/id/volkzap/

TT ผมไม่มี Attributes (ข้อความที่ไม่มี Attributes)

ผมมี Attributes นะผมใช้ style value ผมคือ color: blue;

ผมก็มีลองเอาเมาส์มาชี้ทับผมสิ

Headings
