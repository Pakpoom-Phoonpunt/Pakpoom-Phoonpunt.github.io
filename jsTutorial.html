JS Where To

JS เราสามารถเลือกเขียนได้สองเเบบคือ
1.เขียนในไฟล์ HTML โดยใช้ tag <script>
2.เขียนในไฟล์ .js เเล้วทำการลิ้งมาในไฟล์ HTML

ตัวอย่างการเขียนใน HTML

<head> <script> function myFunction() { document.getElementById("demo").innerHTML = "Paragraph changed."; } </script> </head> <body> <h2>JavaScript in Head</h2> <p id="demo">A Paragraph.</p> <button type="button" onclick="myFunction()">Try it</button> </body> ผลลัพธ์ที่ได้ :

JS Output

ในภาษา JS ผมสามารถดู Output การทำงานของโปรเเกรมที่เราเขียนไว้ได้ด้วยคำสั่งสี่อย่างนี้

คำสั่ง รายละเอียด
innerHTML. จะเเสดง Output ใน HTML Element
document.write(). จะเเสดง Output ใน HTML (หน้าเว็บของเรา)
window.alert(). จะเเสดง Output ใน alert box
console.log(). จะเเสดง Output ใน browser console
(กด F12 เเล้วเลือก console)
ตัวอย่างการใช้คำสั่ง innerHTML. <body> <p>My First Paragraph</p> <p id="demo3">paragraph</p> <script> document.getElementById("demo3").innerHTML = 5 + 6; </script> </body> ผลลัพธ์ที่ได้ :

My First Paragraph

คำตอบล่ะ


จะเห็นว่า tag p ที่มี id = "demo3" ที่ควรเเสดงคำว่า "paragraph" จะถูกเเทนที่ด้วยคำว่า "11" จาก code JS ที่เราเรียกใช้ method .innerHTML = 5 + 6


ตัวอย่างการใช้คำสั่ง document.write(). <body> <script> document.write(5 + 6); </script> </body> ผลลัพธ์ที่ได้ :

ข้อควรระวังสุดๆถ้าเราใช้คำสั่ง document.write หลังจากไฟล์ html ของเราได้ทำการโหลดข้อมูลไปเเล้วคำสั่งจะ delete ข้อมูลที่โหลดมาทำหมดเเล้วเเสดง Output ออกมา

ตัวอย่างที่ควรจะระวัง <body> <p>My first paragraph.</p> <button type="button" onclick="document.write(5 + 6)">Try it</button> </body> มาลองกดดูผลลัพธ์กัน :

My first paragraph.




ตัวอย่างการใช้คำสั่ง window.alert(). <body> <p>My first paragraph.</p> <button type="button" onclick="window.alert(5+6)">Try it</button> </body> ผลลัพธ์ที่ได้ :

My first paragraph.




ตัวอย่างการใช้คำสั่ง console.log(). <body> <script> console.log(5 + 6); </script> </body> ผลลัพธ์ที่ได้ดูได้จากการกด F12 เเล้วเลือกไปที่ console **เฉพาะ browser ที่รองรับ เช่น chrome edge เป็นต้น


JS Statements

JavaScript Statements ประกอบด้วย Values, Operators, Expressions, Keywords และ Comments

ตัวอย่าง
document.getElementById("demo").innerHTML = "Hello World";

Semicolons ;
เราจะใช้ Semicolons ในการเเยกคำสั่งของ JS
เพิ่มเครื่องหมาย Semicolons ที่ส่วนท้ายของคำสั่งปฏิบัติการแต่ละคำสั่ง:
ตัวอย่าง

var a, b, c; // Declare 3 variables a = 5; // Assign the value 5 to a b = 6; // Assign the value 6 to b c = a + b; // Assign the sum of a and b to c

JavaScript White Space
JS เวลา execute คำสั่งจะไม่สนใจการเว้นวรรคเพราะฉะนั้นเราสามารถใช้การเว้นวรรคเพื่อให้ code ของเราอ่านได้ง่ายเเละสบายตา
ตัวอย่าง var person = "Hege"; var person="Hege"; var x = y + z;

JavaScript Line Length and Line Breaks
การเขียน code ที่ดีหนึ่งบรรทัดไม่ควรจะเขียนยาวเกินไป (80 ตัวอักษร) เพราะฉะนั้น JS เราสามารถเขียนในบรรทัดใหม่ได้ตามตัวอย่าง
ตัวอย่าง document.getElementById("demo").innerHTML = "Hello Dolly!";

JavaScript Code Blocks
ใน JS เราสามารถใช้เครื่องหมาย { } ในการจัดกลุ่ม code ของเราได้จะเห็นได้หลักๆคือเมื่อเราเขียน function
ตัวอย่าง function myFunction() { document.getElementById("demo1").innerHTML = "Hello Dolly!"; document.getElementById("demo2").innerHTML = "How are you?"; }


JS Syntax

JavaScript Values
JS จะเเบ่งชนิดของตัวแปรออกเป็นสองชนิด
1. Fixed values (Literals)
2. Variable values (Variables)

JavaScript Literals(Fixed values)
1. Numbers สามารถเขียนได้เเบบมีจุดทศนิยมหรือไม่มีจุดทศนิยม
ตัวอย่าง

document.getElementById("demo_num1").innerHTML = 10.50; document.getElementById("demo_num2").innerHTML = 1001;

2. Strings สามารถใช้ double หรือ single quotes ก็ได้
ตัวอย่าง document.getElementById("demo_str1").innerHTML = "John"; document.getElementById("demo_str2").innerHTML = 'John';


JavaScript Variables
JS เราจะใช้ Keyword var ในการประกาศตัวแปร (Variable)
เเละใช้เครื่องหมาย = ในการ assign values ให้กับตัวแปร ดังตัวอย่าง
ตัวอย่าง var x; x = 6; เป็นการประกาศตัวแปร x เเละ assign value ให้ x = 6


JavaScript Operators
JS จะใช้ตัวดำเนินการทางคณิตในการคำนวณค่า values
ตัวอย่าง (5 + 6) * 10

JavaScript Comments
การ Comment ในภาษา JS จะใช้เครื่องหมาย // สำหรับการ Comment หนึ่งบรรทัด
สำหรับการ Comment หลายๆบรรทัดจะใช้ /* เเละ */ ระหว่าง code ที่เราต้องการจะ Comment


JS Variables

การสร้างตัวแปรของ JS สร้างด้วยการใช้ Keyword var ตัวอย่าง

var x = 5; var y = 6; var z = x + y; จากตัวอย่างสามารถอธิบายได้ว่า

x เก็บค่า 5
y เก็บค่า 6
z เก็บค่า 11 (5 + 6)
