JS Where To

JS เราสามารถเลือกเขียนได้สองเเบบคือ
1.เขียนในไฟล์ HTML โดยใช้ tag <script>
2.เขียนในไฟล์ .js เเล้วทำการลิ้งมาในไฟล์ HTML

ตัวอย่างการเขียนใน HTML

<head> <script> function myFunction() { document.getElementById("demo").innerHTML = "Paragraph changed."; } </script> </head> <body> <h2>JavaScript in Head</h2> <p id="demo">A Paragraph.</p> <button type="button" onclick="myFunction()">Try it</button> </body> ผลลัพธ์ที่ได้ :

ตัวอย่างการเขียนในไฟล์ .js
อย่างเเรกให้เราสร้างไฟล์ .js เเล้วพิม code ตามนี้ function myFunction2() { document.getElementById("demo2").innerHTML = "Paragraph changed."; } เเละมาเชื่อม script เข้ากับไฟล์ HTML ของกันตามนี้ <body> <h2>External JavaScript</h2> <p id="demo2">A Paragraph.</p> <button type="button" onclick="myFunction()">Try it</button> <p>(myFunction is stored in an external file called "js/test.js")</p> </body> <script src="js/test.js"></script> ผลลัพธ์ที่ได้ :
